# Unity Quoridor

Unityで制作しているQuoridor風のボードゲームです。
Unity Editorでプロジェクトを開き、`Standalone.unity` シーンから動作確認できます。

## Overview

このプロジェクトでは、Quoridor風の対戦ボードゲームをUnity上で実装しています。
主な目的は以下です。

- Unityでのボードゲーム実装の練習
- ターン制ゲームの状態管理
- プレイヤー移動・壁配置ルールの実装
- CPU対戦ロジックの検討
- 将来的なUI改善・ゲーム演出追加

## Requirements

- Unity Editor

開発に使用しているUnityバージョンは、プロジェクト設定を確認してください。  
バージョン差異によって警告や再インポートが発生する可能性があります。
